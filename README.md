# 班会寓言故事生成器 - 完整版

一个面向教师/学生群体的寓言故事生成工具，包含完整的授权验证、历史记录、AI集成等功能。

## 功能特点

### 核心功能
- 📝 **授权码验证**：页面初始显示授权码输入框，验证通过后显示主界面
- 📚 **AI故事生成**：通过Vercel Serverless Function调用DeepSeek API生成寓言故事
- 📋 **历史记录**：自动保存生成的故事，支持查看、加载、删除
- 📋 **复制全文**：一键复制完整故事和教学卡片
- 📄 **PDF导出**：导出功能（需集成jsPDF）
- 🖥️ **全屏展示**：适合课堂投屏模式
- 🎨 **专业UI/UX**：现代化界面设计
- ⚙️ **个性化设置**：主题、字体大小、自动保存

### 授权系统
- ✅ 授权码验证（从JSON文件验证）
- 🎁 免费试用3次功能
- 🔄 清除授权功能
- 💾 localStorage记住验证状态

### 历史记录功能
- 📝 自动保存生成的故事
- 📋 按时间倒序排列
- 🔄 加载历史记录到主界面
- 🗑️ 删除单条记录
- 🗑️ 清空全部记录

### AI集成
- 🤖 通过Vercel Serverless Function调用DeepSeek API
- 🔐 使用环境变量存储API Key
- 🔄 失败时自动回退到模拟数据

## 使用方法

### 授权验证
1. 首次打开页面显示授权码输入界面
2. 输入有效的授权码（如：CODE123）
3. 或点击"免费试用3次"按钮（限制3次）
4. 验证通过后显示主界面

### 生成故事
1. 在输入框中输入道理或班级事件（如"不撒谎"、"小明忘记值日还推卸责任"）
2. 点击"生成寓言故事"按钮
3. 查看生成的寓言故事和教学材料
4. 使用工具栏功能进行复制、导出或全屏

### 历史记录
1. 点击"历史记录"按钮查看保存的故事
2. 点击"加载"按钮重新展示历史记录
3. 点击"删除"按钮删除单条记录
4. 点击"清空全部"按钮清除所有历史

## 部署说明

### Vercel部署步骤

1. **创建Vercel项目**
   ```bash
   vercel
   ```

2. **配置环境变量**
   在Vercel项目设置中添加：
   ```
   DEEPSEEK_API_KEY=你的DeepSeek API密钥
   ```

3. **部署项目**
   Vercel会自动检测并部署

### 本地开发

1. **安装依赖**
   ```bash
   npm install
   ```

2. **创建环境文件**
   ```bash
   echo "DEEPSEEK_API_KEY=你的API密钥" > .env.local
   ```

3. **运行开发服务器**
   ```bash
   vercel dev
   ```

### API端点

- **生成故事**：`POST /api/generate`
  请求体：`{ userInput: "用户输入的内容" }`
  响应：生成的完整故事内容
